> A fun and interactive web application where users can create, explore, and rate custom sandwiches built from different breads, ingredients, and styles.

- Vercel (Frontend Hosting)
- Render (API Hosting)

## ✨ Features

- 🥪 Create and edit your own sandwiches
- 🥖 Browse a collection of breads
- ⭐ Like / Dislike voting system
- 💬 Comment system per sandwich
- 📍 Display sandwich origin (country & city)
- 🔍 Search, sort, and filter sandwiches
- 📱 Fully responsive UI (mobile & desktop)

## 📁 Project Structure

- APP Client
- JSON Server

## 🧠 Features Overview

### 🥪 Sandwich System

Each sandwich contains:

- Name
- Nickname (optional)
- Description
- Ingredients
- Bread type
- Location (country / city)
- Votes (likes / dislikes)

### 🥖 Bread System

Each bread includes:

- Name
- Type
- Origin
- Description
- Ingredients list

### 💬 Comments System

Users can:

- Read comments per sandwich
- Add new comments
- View author + date

## 🎨 UI / Design

This project uses:

- Glassmorphism effects (navbar & footer)
- Gradient backgrounds
- DaisyUI components
- Responsive card grid layout
- Clean and modern UX
